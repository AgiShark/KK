# KK
## Studio Coordinate Load Option
![image](https://github.com/jim60105/KK/raw/master/demo/demo.gif)<br>
顧名思義，在Studio的服裝卡讀取那裡，多一個選擇項讓你可以選擇性載入服裝
**1.1版起支援KCOX(Overkay)**

### 需求依賴
BepInEx v4.1.1

### 安裝方式
將KK_StudioCoordinateLoadOption.dll放進BepInEx資料夾

### 下載位置
[Latest Release](https://github.com/jim60105/KK/releases "Latest Release")

遇到任何問題，**請將選項盤全勾即會調用遊戲原始程式碼**
